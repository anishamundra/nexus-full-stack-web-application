## Nexus Market- Premium Online Shopping
This is a professional, server-rendered eCommerce site built with Node.js, Express, EJS and MongoDB Atlas.

## Project Overview
This project is an implementation of full-stack eCommerce application with the following features:
1. a 9 product catelog with responsive CSS grid layout
2. shopping cart with add, update and remove functionality
3. checkout process with order creation
4. server side pricing calculation (subtotal + 13% HST + $20 shipping)
5. MongoDB Atlas for data persistence
6. professional UI with modern design and animation

## Technology Stack
- Backend: Node.js, Express.js
- Database: MongoDB Atlas
- Templating: EJS with partials
- Frontend: HTML5, CSS3, Vanilla Javascript
- Styling: Custom CSS with CSS Variable and modern design

## .env variable
MONGODB_URI=your_mongodb_atlas_connection_string
DB_NAME=MADS-Project
PORT=3000

Run npm run seed for seeding the database
Run npm run dev for starting the development server
